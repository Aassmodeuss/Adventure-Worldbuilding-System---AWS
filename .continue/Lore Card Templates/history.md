# Template - History

Purpose
- Use for historical periods, wars, crises, reforms, diasporas, and pivotal incidents.
- Provide concise context and clear cause->effect chains that inform present-day lore.

Frontmatter
```yaml
---
id: history:unification-war
type: history
name: Unification War
parent: history:turian-civil-era   # optional: use only if older history is required context
see_also: []
---
```

Card template
```markdown
[ENTRY: {Name}]

#### [History]
**Name:**  
**Summary:** 1-2 sentences on what happened, when, and why it matters now. Write this as a standalone section; do not rely on Full Description.  
**Full Description:** 4-12 sentences, organized under brief headings for clarity (see below). Write this as a standalone section; do not rely on Summary.  
**Tone Keywords:** 3-5 anchors.  
**Key Dates:** concise list or range (e.g., 412-418 AS).  
**Primary Theater(s):** realms/regions/locations most affected.  
**Related Characters:** comma-separated character card names (leaders, instigators, witnesses).  
**Related Factions:** comma-separated faction card names.  
**Related Races:** comma-separated race card names, if central to events.  
**Related Places:** comma-separated place card names (world/realm/region/biome/location).  
**Potential Gameplay Hooks:** 1-3 concise bullets tying past events to present stakes.  

  

[END: {Name}]
```


Pre-save checklist
- Parent only when prerequisite context is required; otherwise omit.  
- Related Characters/Factions/Races/Places populated when applicable.  
- Summary states what, when, and why it matters; Full Description uses headings.  
- Wrappers and Name field match exactly.
