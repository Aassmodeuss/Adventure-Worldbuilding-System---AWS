# Template - Being: Faction

Purpose
- Use for organized groups, orders, syndicates, councils.

Frontmatter (optional parent)
```yaml
---
id: faction:brightvale-council
type: faction
name: Brightvale Council
parent: region:westwood           # optional: realm/region/location if seated there
see_also: []
---
```

Card template
```markdown
[ENTRY: {Name}]

#### [Faction]
**Name:** 
**Aliases:**  
**Summary:** 1-2 sentences on identity and significance. Write this as a standalone section; do not rely on Full Description. Include the card's title name at least once.  
**Full Description:** 4-10 sentences on history, structure, aims, operations, and conflicts. Write this as a standalone section; do not rely on Summary. Include the card's title name at least twice (natural usage; avoid repetition).  
**Tone Keywords:** 3-5 anchors.  
**Aligned Characters:** comma-separated character card names  
**Known Locations:** comma-separated location card names  
**Potential Gameplay Hooks:** 1-3 concise bullets, potential quests, events, or interactions involving this faction.  


[END: {Name}]
```

Pre-save checklist
- Known Locations and Aligned Characters populated.  
- Parent set only if the faction has a clear seat in the tree.
