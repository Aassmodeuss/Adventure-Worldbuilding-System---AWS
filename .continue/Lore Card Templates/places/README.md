# Lore Card Templates - Places

Available templates
- world.md - top-level world cards
- realm.md - major realms within a world
- region.md - sub-areas within a realm
- biome.md - ecological zones
- location.md - specific settlements and sites
- point.md - rooms or micro-sites within a location
- place_concept.md - umbrella geographical concepts used for parallel links

Usage
- Copy the frontmatter and card template into your lore file.  
- Follow the Delta detail rules to avoid repeating parent information.  
- Add a Links block for Foam: Parent and See also.
